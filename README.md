# Online Job Portal

## 📌 Project Description
Online Job Portal is a web-based application developed using Java, JSP, Servlets, and MySQL. It helps job seekers search and apply for jobs while recruiters can post jobs and manage applications.

## 🚀 Features
- User Registration & Login
- Recruiter Registration & Login
- Post New Jobs
- Search Jobs
- Apply for Jobs
- Manage User Profile
- View Applied Jobs
- Recruiter Dashboard

## 🛠️ Technologies Used
- Java
- JSP
- Servlets
- HTML
- CSS
- JavaScript
- MySQL
- Apache Tomcat
- NetBeans IDE

## 📂 Project Structure
- src/ - Java source code
- web/ - JSP, HTML, CSS files
- nbproject/ - NetBeans project files
- build.xml - Build configuration

## ▶️ How to Run
1. Install Java JDK.
2. Install NetBeans IDE.
3. Install Apache Tomcat.
4. Import the project into NetBeans.
5. Configure the MySQL database.
6. Run the project on Tomcat.
